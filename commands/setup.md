Help the user set up TrueFoundry AI Gateway for their project:

1. Check if TFY_BASE_URL and TFY_API_KEY are configured
2. If not, guide them to get credentials from their TrueFoundry dashboard
3. Detect which coding agents are installed (Claude Code, Codex, Cursor)
4. Offer to configure each agent to route through the gateway
5. Run a smoke test to verify connectivity
6. Print a summary of what was configured

Use the onboard skill for the full interactive flow.
